# Improving Generative AI Systems through Prompt Optimization

## Problem Summary
Generative AI models like GPT are powerful but suffer from:
- Prompt sensitivity
- Poor long-text handling
- Inconsistent outputs

## Approach
- Prompt optimization
- Chunk-based processing
- Evaluation using BLEU and ROUGE
